# 🤖 LazyMan Daily Tech News Bot

一个专为科技爱好者设计的自动化新闻推送机器人。它专注于追踪 **马斯克 (Elon Musk)** 与 **特朗普 (Donald Trump)** 的最新动态，并自动为英文资讯提供中文描述，全程**无需任何付费 API**。

## 🎯 核心功能

- **🚀 马斯克专栏**：深度抓取 Tesla, SpaceX, Neuralink 及 X.ai 的最新动态（每日上限 15 条）。
- **🏛️ 特朗普追踪**：实时监控特朗普及其相关政治、政策新闻（每日上限 15 条）。
- **🌍 零成本中文描述**：采用内置词库匹配技术，自动为英文标题生成中文关键词标注。
- **📊 智能属性分类**：基于关键词算法，将新闻自动归类至 AI、宇航、金融、安全等 10 个核心维度。
- **⏰ 全自动化运行**：利用 GitHub Actions 实现每日定时抓取与推送，无需部署虚拟服务器。
- **🚀 多渠道推送**：支持微信（Server酱）、Telegram 机器人等多种主流推送方式。

## 📋 功能特性

### 1. 新闻抓取与分拣
- **优质源集成**：预设 NYT Technology、TechCrunch、The Verge 等权威媒体。
- **优先级排序**：马斯克与特朗普相关资讯拥有最高权重，确保排在简报最前端。
- **智能去重**：自动过滤重复标题，确保每份简报内容精炼纯净。

### 2. 翻译与展示 (No-API 模式)
- **关键词高亮**：通过内置 `TRANS_MAP` 自动替换术语。
- **双语友好**：保留原始标题的同时提供中文核心点概述。
- **Markdown 格式**：生成的简报排版精美，适合手机端查看。

## 🚀 快速上手 (Deployment)

### 第一步：创建仓库
1. 在 GitHub 上新建一个仓库（建议设为 **Private**）。
2. 将项目中的 `main.py` 放在根目录。
3. 创建文件夹 `.github/workflows/` 并将 `daily.yml` 放入其中。

### 第二步：配置密钥 (Repository Secrets)
前往仓库的 **Settings -> Secrets and variables -> Actions**，点击 **New repository secret** 添加以下配置（根据需求选择）：

| 变量名 | 说明 |
| :--- | :--- |
| `SC_KEY` | [Server酱](https://sct.ftqq.com/) 的 SendKey（用于微信推送） |
| `TG_TOKEN` | Telegram Bot 的 API Token |
| `TG_CHAT_ID` | 接收消息的 Telegram 用户 ID 或频道 ID |

### 第三步：激活自动化
1. 点击仓库上方的 **Actions** 选项卡。
2. 在左侧选择 **Daily News Bot** 工作流。
3. 点击 **Run workflow** 手动触发第一次测试。
4. 此后，机器人将于每天北京时间 **09:00** 自动发送简报。

## 🛠️ 如何自定义 (Customization)

### 修改关键词或分类
您可以直接编辑 `main.py` 中的以下内容：
- **`TRANS_MAP`**: 添加或修改你想要翻译的专有名词。
- **`CATEGORIES`**: 调整 10 个分类的关键词权重或名称。
- **`RSS_SOURCES`**: 增加你个人偏好的新闻订阅源。
